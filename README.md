# Ankarton

Ankama account's creator

## Installation

```
$ npm install -g ankarton
```

## Usage

```
$ ankarton --out ACCOUNTS.txt --proxy PROXY-LIST.txt --total ACCOUNT_NUMBER
```

## Development

 * Prerequisites

   ```
   $ npm install
   ```

* Lint the ts files

    ```
    $ npm run lint
    ```
    or to fix some errors automatically
    ```
    $ npm run lint:fix
    ```  


* Build the js files

    ```
    $ npm run build
    ```
